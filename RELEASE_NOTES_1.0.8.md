# Release notes for v1.0.8 (GitHub release)

Use the text below when creating the **v1.0.8** release on GitHub (Releases → Draft a new release → tag `v1.0.8`).

---

## Release title
**v1.0.8** — App Store & accessibility

## Description (copy into the release)

**Free download for macOS (same app is also on the Mac App Store as a paid download).**

### What's new in 1.0.8
- **App Store:** Sandbox, encryption declaration, accessibility (VoiceOver, dark mode, Accessibility URL)
- **Export compliance:** App uses only standard HTTPS (no non-exempt encryption)
- **Documentation:** Updated README (free on GitHub, paid on App Store), Xcode Cloud and App Accessibility guides
- **Support:** GitHub Issues and [website](https://itsmonoyo.iamjarl.com) with accessibility info

### Requirements
- macOS 11.0 or later
- Up to 50 WAV files per batch, 100 MB max per file

### Install
1. Download **It's mono, yo!-1.0.8.zip** (or the `.app` from the assets below)
2. Unzip if needed and move the app to Applications
3. First launch: right-click → Open (Gatekeeper)
